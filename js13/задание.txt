Создать страницу регистрации пользователя, используя HTML, CSS и JavaScript, 
которая требует от пользователя ввести данные в несколько полей. Каждое поле 
должно быть проверено на стороне клиента с использованием регулярных выражений 
(RegExp) для обеспечения соответствия определённым требованиям. Ваша задача 
включает в себя дизайн формы, стилизацию и написание скрипта для валидации 
каждого поля в реальном времени.

Требования к полям формы:
Имя пользователя:

Должно содержать только буквы.
Длина от 3 до 16 символов.
Email:

Должен соответствовать стандартному формату электронной почты.
Пароль:

Минимум 8 символов.
Минимум одна заглавная буква, одна строчная буква и одна цифра.
Может содержать специальные символы (!@#$%^&*).
Подтверждение пароля:

Должно точно совпадать с введённым паролем.
HTML и CSS:
JavaScript и регулярные выражения:
Напишите функции валидации для каждого поля, используя регулярные выражения.
Показывайте сообщения об ошибках рядом с соответствующими полями, если введённые данные 
не соответствуют требованиям.
Форма должна быть отправлена только в том случае, если все поля успешно прошли валидацию.
Дополнительные вызовы:
Реализуйте динамическую валидацию поля в реальном времени (т.е., проверка каждого 
поля при изменении его значения, без необходимости нажимать кнопку отправки формы).
Добавьте анимации для всплывающих сообщений об ошибках.
Сделайте форму адаптивной для различных размеров экранов.
